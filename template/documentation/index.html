Moje. is a awesome vCard, one page HTML / CSS template based on Bootstrap 3 framework!. Thanks to responsive design it is going to look awesome on any device!

HTML Structure

Whole template is based on Bootstrap 3 framework. Everything about this framework you will wind here.

Below you see the structure of the template:

<!-- Header  -->
<header id="nav-content" class="vertical-nav nav-white">

    << CONTENT GOES HERE >>

</header>
<!-- Header / End -->

<!-- Page Content -->
<div id="page-content">

    ...
    <!-- Section -->
    <section id="sections-name">

        << CONTENT GOES HERE >>

    </section>
    ...

</div>
<!-- Page Content / End -->

Every section is divided according to Bootstrap Grid System.

Important Classes

This template includes also some additional classes like:

.content - this class adds top padding
.black - use this class to section with black background
.dark  - use this class to section with dark background - it changes font color to white
.padded  - this class adds 60px padding
.padded-short  - this class adds 30px padding
.padded-vertical  - this class adds 70px top and bottom padding
.short - this class removes bottom margin
.medium - this class changes bottom margin to 10px
.long - this class changes bottom margin to 50px

Configuration

Photo Alpha

To customize transaprency of your photo you need to use similar code like in the background image in #profile section

<div id="profile-bg" class="editable-alpha" data-alpha="40" style="opacity: 0.4;">

    <img alt="" src="images/backgrounds/profile-bg.jpg"></img>

</div>

.editable-alpha - required class

data-alpha="image opacity in %" - required attribiute

 

Google Map

To configure your location you have to edit two lines in js/GoogleMap-dark.js or js/GoogleMap-light.js file

var yourLatitude = 40.758895;
var yourLongitude = -73.985131;

You need to use special toolto find your latitude and longitude (f.e http://itouchmap.com/latlong.html)

js/GoogleMap-dark.js - use it if you want to use light Map Theme

js/GoogleMap-dark.js - use it if you want to use darkMap Theme

 

Video Background

To add video background to the Profile section you have to remember to add configuration script.

/* Video Background */
$("#profile-bg").hide();
$(".player").mb_YTPlayer();
if( /Android|webOS|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(navigator.userAgent) ) {
    $("#wrapper_mbYTP_profile-bg-player").hide();
    $("#profile-bg").show();
}

To configure your video you need to edit this code:

<!-- Video Background Player -->
<div id="profile-bg-player" class="player" data-property="{videoURL:'http://youtu.be/PUT_HERE_VIDEO_ID',containment:'#profile',startAt:16,stopAt:200,mute:true,autoPlay:true,loop:true,showControls:false,opacity:0.35}"></div>

You need to put yours video ID into this part "PUT_HERE_VIDEO_ID" from the above code.

Both of that codes was added to index-video.html and index-clean-video.html pages.

Important! Video Background does not work in offline version.

 

Resume Dimmed Effect

To remove this effect please remove dimmed-effect class this part of code in Resume section:

<div class="content dimmed-effect padded-short">
